The College Housing Project collects public data from the Hays County Appraisal District, the San Marcos Police Department, and other sources. This page provides links to the original data files used in the project, as well as an explanation of the processing involved.

Data Files

Included below are original data files obtained from public agencies and processed for this project. Currently, property records are formatted as delimited text files, and police, fire and EMS records are formatted as .xslx and .csv files. All of these files can be opened in Excel or similar spreadsheet software.

Notes

Police Records

The incident blotters included here contain all police calls, regardless of whether they resulted in a police report or police action.

Fire and EMS Records

Fire and EMS records are collected together from the San Marcos Fire Department. We consider a record to be an EMS call if the description begins with "EMS."

Data Processing

Inclusion of Properties

Properties are included in the database based on records from the Hays County Appraisal District. Only properties in the 78666 area code and designated using the B1 land type code (residential: multi-family) are included.

Locations and Distances

Information related to distances is calculated using coordinates obtained using Google's Geocoding API. Because data such as police reports are entered manually, some addresses cannot be converted to coordinates and thus are not included. Because some addresses are reported as highways exits or consist only of street names, the locations and distances reported should be considered approximate.
